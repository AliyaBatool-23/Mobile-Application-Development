# HireNest
An Android app that connects students with recruiters using AI-powered CV ranking.
## Installation
1. Clone the repository
2. Add 'google-services.json' from Firebase Console to '/app' folder
3. Add Gemini API key in 'ApplicantsActivity.java'
4. Add Cloudinary credentials in 'MainActivity.java'
5. Run on Android device
